Si desea ver la base de datos en workbench abra el achivo ubicando en la siguiente ruta:

/srv/wwww/html/eduline/documentacion/baseDatos/eduline.mwb Las siguientes tablas son incluídas dentro de la base de datos:

  • Tabla de roles
  • Tabla de usuarios
  • Tabla de acciones
  • Tabla de areas de conocimiento
  • Tabla de materias
  • Tabla de cursos

La tabla de roles

La tabla de roles permite agrupa a los usuarios de acuerdo a los permisos que estos tienen dentro de eduline. Cada usuario al ser registrado deberá tener un rol y en base a eso el sistema podrá determinar que acciones se le permiten ejecutar.

Los campos

Nombre del campoDescripciónTablas relacionadas¿Valor único?valores estaticos
id_rolDe una clave numérica única a cada rol usuarios, accionesSi
nombreEs el nombre del rol que describirá principalmente la característica de los usuarios S/RSi
descripcionPermite dar mas detalles sobre las características de los usuarios que tengan asignado este rol S/RNo

Relaciones a otras tablas

Tabla relacionadaTipo de relacióncampo relacionadoDescripción
usuariosUno a muchosid_rolCada usuario sabe a que rol pertenece, muchos usuario pueden pertenecer a un rol
accionesMuchos a muchosid_rol Pueden haber muchas acciones asignadas a un rol a una acción asignada a muchos roles. Para hacer la relación muchos a muchos se utiliza la tabla roles_acciones. Con este relación el sistema puede determinar que acciones son permitidas a un usuario que se loguea con determinado rol.
